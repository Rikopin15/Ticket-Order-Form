Ticket Ordering Website

A simple multi-page concert ticket ordering website built using HTML and CSS.  
This project allows users to view concert information and submit ticket orders through a form powered by Formspree.

---

## Features

- Multi-page website
- Navigation bar
- Concert information page
- Ticket ordering form
- Seat selection options
- Payment method selection
- Form submission using Formspree
- Styled using external CSS

---

## Pages

### Home Page (`index.html`)
Contains:
- Concert announcement
- Tour information
- Navigation menu
- "Order Now" button linking to the order page

### Order Page (`order.html`)
Contains:
- User information form
- Seat selection dropdown
- Payment method dropdown
- Form submission system

---

## Technologies Used

- HTML5
- CSS3
- Formspree
